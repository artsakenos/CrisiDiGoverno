# crisidigoverno files
Contiene i file utili a lanciare la crisi di governo.
* La grammatica, in quel formato similBNF con il generatore che avevo fatto e che hai sull'altro progetto che avevo condiviso (con le lib di facebook)
* Il model controller, giusto per curiosità, che pusha le robe dentro alla view
* La View che è nient'altro che una UltraPage con dei placeholder (per curiosità i placeholder possono essere messi in un paio di modi, ma in realtà tutta la pagina, o meglio tutto un sito può essere modellato dalle librerie a piacimento).
